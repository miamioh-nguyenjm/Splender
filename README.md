How to install:
1. Download the zip in releases
2. Extract anywhere (it's portable)
3. Make sure you have installed the Visual C++ redistributables: https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170. Or the auto installer for them all: https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/.
4. Run splender_gl.exe
